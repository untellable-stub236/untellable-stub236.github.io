# CeloHT Agent Network

The CeloHT Agent Network is a structured community initiative designed to support digital adoption, education, and financial inclusion.

## Agent Roles
- Community Educator  
- Merchant Support Specialist  
- Digital Payments Facilitator  
- Environmental Program Assistant  

## Responsibilities
- Educate users  
- Support merchants  
- Promote safe digital practices  
- Assist in reforestation initiatives
